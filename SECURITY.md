# Security Policy

## Secrets

Do not commit Discord webhook URLs.

The webhook URL should live in:

```bash
~/.config/codex-discord-notifier/env
```

The repository only includes `env.example`.

## Reporting Security Issues

Please open a private security advisory on GitHub if available, or contact the repository owner directly.

Avoid posting live webhook URLs, Codex logs, private repository names, or sensitive thread content in public issues.
